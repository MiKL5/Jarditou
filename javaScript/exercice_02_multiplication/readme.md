> ### ***Cet exercice est en javascript***


* Une fonction affiche une table de multiplication.
* Prendre un paramètre qui permet d'indiquer quelle table afficher.
* Par exemple, TableMultiplication(7) doit afficher :
    * *1 x 7 = 7*
    * *2 x 7 = 14*
    * *3 x 7 = 21 et cetera,*
